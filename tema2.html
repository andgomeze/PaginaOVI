Consultas a una base de datos

Ya que hemos accedido al servidor y y conectado con nuestra base de datos podemos recuperar datos para luego mostrarlos al usuario final.

Para realizar esto necesitamos hacer la consulta como la entiende mysql, por ejemplo “SELECT CAMPO1,CAMPO2,.. FROM TABLA”, y luego mediante la función mysql_query enviar la sunsulta al servidor de bases de datos.

mysql_query($consulta,$conexion)

Para mostrar los resultados usamos mysql_fetch_array, como su nombre lo indica devuelve un array donde cada campo recuperado es un ítem de este y podemos acceder a él por su nombre:

$fila = Mysql_fetch_array($consulta) // Se asocia el array a la variable

Con un while le indicamos a php que muestre los resultados mientras se cumpla con todos los registros de la consulta.

While ($fila = Mysql_fetch_array($consulta){ echo “Encabezado”.$fila[“Campo1”].$fila[“Campo2”] }

Inserción

Así como en la consulta primero hay que indicar cuales son los datos que se desean ingresar, y posteriormente ejecutar la función mysql_query.

$insertar= “INSERT INTO TABLA (Campo1) VALUES (‘DATO’)”
mysql_query($insertar,$conexion)

Eliminación

De igual manera que las anteriores solo se indica la indtruccion de lo que se desea elimianar con la misma función mysql_query

$elimina= “DELETE FROM TABLA WHERE Campo = ‘Condicion’”
mysql_query($elimina,$conexion)
