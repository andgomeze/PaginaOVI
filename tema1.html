Conexión con una base de datos

Para realizar la conexión con la base de datos debemos tener en cuenta lo siguiente:

  • Conocer el nombre de la base de datos.
  • Tener credenciales de acceso a esta.
  • Saber el nombre del servidor donde está alojada la base.

    • Teniendo claro esto se procede a realizar el código que nos permitirá enlazar a php con mysql. La función principal que sirve a este propósito es mysql_connect, cuya sintaxis es la siguiente:

    mysql_connect(string $server, string $username, string $password)

    Donde server es una cadena que indica el nombre del servidor donde se encuentra la base de datos, username es el usuario con el que nos conectaremos al servidor, password la contraseña de acceso. Una vez conectados al servidor debemos indicar a cuál base de datos deseamos acceder mediante la función mysql_select_db:

    mysql_select_db($databasename,$conexion)
